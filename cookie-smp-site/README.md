# Cookie SMP Website (Student Project)

This is a 6-page multi-page website template for a Minecraft Vanilla+ SMP server.

## Pages
- index.html (Home)
- about.html
- features.html
- events.html (includes a table)
- gallery.html (includes an embedded video)
- join.html (includes a form)

## Assets
- assets/css/styles.css (shared styles)
- assets/js/main.js (mobile menu toggle)
- assets/images/*.svg (placeholder images)

## GitHub Pages
Upload these files to a GitHub repo and enable GitHub Pages.
Open `index.html` as the homepage.
