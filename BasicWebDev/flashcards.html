Web Design Flashcards

What does HTML stand for?
HyperText Markup Language
What tag creates a link?
<a> tag
What does CSS control?
The style and layout of a webpage
What property changes text color?
color
What does display: flex do?
Creates a flexible layout
What property adds space inside an element?
padding
What selector targets all paragraphs?
p
What does hover do?
Applies styles when the mouse is over an element
